# Image Background Remover - MVP 需求文档

**项目名称**: BackgroundRemover.dev（可改）  
**定位**: 免费在线图片背景移除工具，面向设计师、电商卖家、内容创作者  
**起步成本**: < $5，零服务器成本  

---

## 一、产品定位

### 目标用户
1. **个人用户**：偶尔需要移除图片背景，不想下载软件
2. **小型商家**：电商产品图修图，预算有限
3. **开发者**：需要简单的在线工具快速处理图片

### 核心价值
- ✅ 完全在线，无需下载安装
- ✅ 1 秒钟移除背景，操作简单
- ✅ 免费额度够用，付费无水印高清
- ✅ 隐私保护，图片不存储，处理完立即删除

### 差异化
- 比 remove.bg 便宜，比免费工具无水印广告少
- 界面简洁，没有弹窗骚扰
- 移动端友好，手机也能用

---

## 二、核心功能 MVP

### 1. 首页功能
- [ ] 大拖拽上传区域（支持点击选择）
- [ ] 支持常见格式：JPG、PNG，最大文件限制 10MB
- [ ] 自动处理，不用用户调整参数
- [ ] 处理前后对比预览（左右拖拽滑块）
- [ ] 一键下载处理后的图片（PNG 透明背景）

### 2. 限流与付费
- [ ] 游客：每天免费 3 次，低分辨率，弱水印（可选）
- [ ] 登录用户（Google/GitHub）：每天免费 5 次
- [ ] 单次付费：$0.99 无限次当天使用
- [ ] 月订阅：$2.99 无限次/月，支持 API 调用（预留）

### 3. 技术架构（优化版）
```
┌─────────────────────────────────────────────────┐
│  Cloudflare Pages (Next.js 前端托管)            │
└────────────────┬────────────────────────────────┘
                 │
         ┌───────▼──────────┐        ┌───────────────────┐
         │ 浏览器端处理     │  ┌────▶ │  Remove.bg API    │
         │ - 压缩           │  │      │  - 付费按调用量   │
         │ - 预览           │  │      └───────────────────┘
         │ - 不落地存储     │  │
         └──────────────────┘  │
                              │  全程不存储用户图片，内存处理
                              │
         ┌────────────────────▼┐
         │ Cloudflare Worker   │
         │ - 转发 API 请求      │
         │ - 限流 & 计数        │
         │ - 隐藏 API Key       │
         └──────────────────────┘
```

### 4. 技术细节
- **前端**: Next.js 14 + TailwindCSS + ShadCN UI
- **托管**: Cloudflare Pages（免费）
- **API 代理**: Cloudflare Worker（免费）
- **图片处理**: Remove.bg API（按调用量付费，起步免费）
- **认证**: Auth.js（NextAuth）支持 Google/GitHub 登录（可选 MVP 后加）
- **支付**: Stripe 订阅 + 单次支付（预留接口）

### 5. 非功能需求
- [ ] LCP < 1.5s，全球访问快（Cloudflare CDN 搞定）
- [ ] 移动端适配，手机上传没问题
- [ ] 隐私声明：图片不存储，不用于训练
- [ ] 错误处理：文件太大/API 限额给明确提示

## 五、风险控制

| 风险 | 应对方案 |
|------|----------|
| Remove.bg API 涨价 | 提前预留自有模型接入接口，流量大了切自托管 |
| 被刷 API 额度 | IP 限流 + 免费额度限制，付费后放开 |
| 成本过高 | 早期用免费额度验证，有收入再扩容 |
| 合规问题 | 明确隐私政策，不存储用户图片，规避内容风险 |
